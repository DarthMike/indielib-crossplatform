typedef int IND_InitializationMode

g_debug

Using IndieLib::init() method, the library can be initialized with the modes: IND_DEBUG_MODE (debug mode) , IND_RELEASE_MODE (release mode).

IND_DEBUG_MODE starts the library in debug mode. In this mode at the end of the execution a file named debug.txt is generated. It contains details about the project execution. Here you can verify if the resources are correctly loaded/freed. Please check this file if any problem happen after the execution of an IndieLib application.

IND_RELEASE_MODE is the mode that should be chosen for a finished project.
